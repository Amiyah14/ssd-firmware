PREREQUISITES:

HOW TO USE:
1. Download the SoftPaq .EXE file to a directory on your hard drive.
2. Execute the downloaded file and note where the directory where the files are extracted to.
3. Open an administrative command prompt
3a. The output of the utility is 160 characters wide - in order to see all information in the format as intended, it is suggested to resize your command prompt window to a minimum of 160 characters wide
4. Navigate to the directory where the SoftPaq files were extracted to.
5. Type "FWUpdate.bat" and press "Enter"
6. The workstation MUST be shut down after the firmware update completes to activate the new firmware
